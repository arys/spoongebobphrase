## SpongeBob Phrase (RU)

Мини-сайт на Next.js: пользователь вводит фразу на русском, приложение ищет её в `.srt` субтитрах и показывает совпадения, а также **ссылку на YouTube с нужного таймкода** + встроенный плеер.

## Запуск

Установите зависимости и запустите dev-сервер:

```bash
npm i
npm run dev
```

Откройте `http://localhost:3000`.

## Данные

- `app/data/index.json` — конфиг эпизодов: YouTube URL + путь к `.srt`.
- `app/data/*.srt` — субтитры (русские).

## API

- `GET /api/search?q=...&episode=s1e1`
  - Возвращает до 50 совпадений.
  - Для каждого совпадения отдаёт `youtubeUrl` (с `t=`) и `embedUrl` (с `start=`).
